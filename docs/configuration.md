# Configuration

Use `DJANGO_FORGE` in Django settings.

```python
DJANGO_FORGE = {
    "brand_name": "Forge Admin",
    "brand_logo_text": "FORGE",
    "brand_tagline": "Modern Django operations panel",
    "accent_color": "blue",
    "default_theme": "system",
    "show_sidebar_search": True,
    "enable_command_bar": True,
}
```
